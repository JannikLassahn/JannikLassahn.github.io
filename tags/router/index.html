04. Feb 2022
#IIS  #SPA  #router  #cache 
This is a short guide for setting up an Internet Information Services (IIS) site that serves a Single Page Application (SPA). There are three obstacles to overcome here: How to serve a SPA with client-side routing How to serve a non-cached, always up to date version of the SPA How to prevent IIS from shutting down automatically Each problem will be addressed in this post, with additional information below the config.
